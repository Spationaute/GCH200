Le cours GCH200: Phénomènes d'échanges I est un cours de 3ème session, dans les programmes de génie chimique et de génie biotechnologique de l'Université de Sherbrooke. Le cours est basé sur le texte "Transport Phenomena", des professeurs Bird, Stewart et Lightfoot de l'Université du Wisconsin à Madison. Nous couvrons dans ce premier cours les chapitres 1,2,6,7,9,10,17 et 18. Chaque Notebook vise à expliciter les développements montrés dans le texte en donnant à tous les outils de calcul provenant du vaste "écosystème" du langage python. Les notebooks Jupyter permettent de mettre dans le même document texte, équations, manipulations mathématiques, solutions analytiques et numériques et illustration graphique dans un document connecté sur le web. Les packages python SYMPY NUMPY COOLPROP THERMO FLUIDS MATPLOTLIB notamment seront utilisés afin de permettre la manipulation des concepts physico-chimiques impliqués dans la discipline des phénomnres d'échanges.

Pierre Proulx, 2018.
